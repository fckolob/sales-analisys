# Overview

This program shows how to implement data analysis and data frames manipulation using Pandas and Matplotlib to analyze and display data. 

This program analyzes a dataset from an ecommerce sales report that categorizes the sold products by category size and color and helps to see the trends in the sales and what is more demanded. This is the link to the sales report dataset: https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data?resource=download&select=Sale+Report.csv 

I wrote this code to demonstrate how to implement various Python technologies like Pandas, Matplotlib, Ttkbootstrap and Tkinter to work together to analyze and show data.

# Development Environment

This program was developed using Visual Studio Code.

The libraries used are: Pandas (to work with data frames and manipulate them), Matplotlib (to create and display charts), Tkkbootstrap (to create a modern UI) and Tkinter (for some parts of the UI, for example the dialog to load a dataset file).
